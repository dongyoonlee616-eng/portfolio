# 💼 Dong-Yoon Lee Portfolio

개발자를 꿈꾸는 학생 **이동윤(Dong-Yoon Lee)**의 포트폴리오 사이트입니다.

HTML, CSS, JavaScript를 기반으로 제작되었으며,  
저의 성장 과정, 기술 스택, 활동 기록, 프로젝트를 정리하는 개인 포트폴리오입니다.

---

## 🌐 배포 주소

🔗 https://dongyoonlee616-eng.github.io/portfolio/
📈 최신 버전 : 1.0.3

---

## ✨ 주요 기능

- 자기소개 및 성장 과정 소개
- 기술 스택 소개
- 수상 및 활동 기록 소개
- 프로젝트 목록 페이지
- 프로젝트 상세 페이지
- Contact 페이지
- 404 페이지
- 반응형 웹 디자인
- SEO 최적화
- Open Graph 지원
- GitHub Pages 배포

---

## 📄 페이지 구성

| 페이지 | 설명 |
|---|---|
| Home | 메인 소개 페이지 |
| Journey | 개발을 배우며 성장한 과정 |
| Skills | 현재 공부하고 있는 기술 스택 |
| Awards | 수상 및 활동 기록 |
| Projects | 제작한 프로젝트 목록 |
| Project Detail | 프로젝트 상세 설명 |
| Contact | 연락 정보 |
| 404 Page | 잘못된 주소 접근 시 표시되는 페이지 |
