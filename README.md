# 💼 Dong-Yoon Lee Portfolio

개발자를 꿈꾸는 학생 **이동윤(Dong-Yoon Lee)**의 포트폴리오 사이트입니다.

HTML, CSS, JavaScript를 기반으로 제작되었으며,
다양한 프로젝트와 성장 과정을 기록하는 개인 포트폴리오입니다.

# 🌐 배포 주소

# ✨ 주요 기능

- 자기소개 및 성장 과정 소개
- 기술 스택(Skills) 소개
- 수상 및 활동(Awards) 소개
- 프로젝트(Project) 소개
- 프로젝트 상세 페이지
- Contact 페이지
- 404 페이지
- 반응형 웹 디자인
- SEO 최적화
- Open Graph(OG) 지원
- GitHub Pages 배포
