Overview

프로젝트 소개

이 사이트는 이동윤이라는 사람을 소개하고, 개발자로 성장해가는 과정을 보여주는 포트폴리오 웹사이트입니다. Home, Journey, Skills, Awards, Projects, Contact 페이지로 구성되어 있으며, 앞으로 만들어갈 프로젝트들을 기록하는 공간으로 발전시킬 예정입니다.

Purpose

제작 목적

한국디지털미디어고등학교 입학 목표와 개발자라는 꿈을 보여주기 위해 제작했습니다. 단순한 자기소개 페이지가 아니라, 제가 어떤 분야에 관심이 있고 어떤 과정을 통해 성장하고 있는지를 보여주는 포트폴리오를 만드는 것이 목표였습니다.

Tech Stack

사용 기술

HTML

웹사이트의 구조와 페이지 내용을 구성했습니다.

CSS

전체 디자인, 반응형 레이아웃, 카드 디자인, 애니메이션 효과를 구현했습니다.

JavaScript

타이핑 효과, 로딩 화면, 스크롤 등장 애니메이션, 커스텀 마우스 효과 등을 구현했습니다.

GitHub Pages

완성된 웹사이트를 온라인에 배포했습니다.

Problem Solving

어려웠던 점과 해결 방법

문제점 1

CSS 코드가 2000줄 이상으로 길어지면서 버튼, 카드, 애니메이션 코드를 찾고 수정하기 어려워졌습니다.

해결 방법

CSS를 base, effects, layout, hero, components, pages, animations, icons, responsive 파일로 분리하여 역할별로 관리할 수 있도록 리팩토링했습니다.

문제점 2

아이콘 시스템을 적용하는 과정에서 Contact 카드 레이아웃이 깨지거나, 프로젝트 카드가 좁은 영역에 들어가면서 보기 좋지 않게 표시되는 등 여러 디자인적인 문제가 있었습니다.

해결 방법

카드 전체에 무리하게 공통 스타일을 적용하지 않고, 아이콘과 카드 레이아웃을 분리해서 관리하도록 수정했습니다. 또한 프로젝트 목록 페이지에서는 대표 프로젝트 카드를 크게 보여주고, 추가 예정 프로젝트 카드는 아래에 배치하는 방식으로 레이아웃을 개선했습니다.

What I Learned

배운 점

이 프로젝트를 만들면서 HTML로 페이지 구조를 설계하고, CSS로 디자인과 반응형 레이아웃을 구성하는 방법을 익혔습니다. JavaScript를 이용해 타이핑 효과, 로딩 화면, 스크롤 등장 애니메이션, 커스텀 마우스 효과를 구현하면서 웹사이트에 동적인 요소를 추가하는 방법도 배웠습니다.

특히 CSS 코드가 길어졌을 때 파일을 역할별로 분리하고 리팩토링하는 과정의 중요성을 알게 되었습니다.
